# Présentation Orale d'un réseau social au choix

## I. Modalités

Présenter un réseau social de votre choix devant la classe par groupe de deux. 

La présentation devra durer environ une dizaine de minutes.

Un diaporama pourra être crée et projeté via mon ordinateur.

## II. Contenu

Ci-dessous une liste non exhaustive de questions auquels vous pouvez répondre dans votre présentation :

-> Qu'est-ce qu'un réseau social ?

-> Description du réseau choisi :
    + Nombre d'utilisateur ?
    + Quelle cible auprès des utilisateurs ?
    + Usage : pourquoi l'utilise t-on ?
    + Évolution du réseau ?

-> Quel type de lien ont les utilisateurs entre eux ?

-> Quel type de contenu est mis en avant sur la plateforme ?

-> Comment le réseau fait-il pour gagner de l'argent et l'entreprise être pérenne ?

-> ...
